== Installation ==

Extract the zipped package downloaded from WordPress.org to your desktop, in the extracted package you will find lemur-slider.zip file which is the WordPress plugin.

You can install the plugin in two ways:

FTP: Extract lemur-slider.zip file and upload the extracted folder to /wp-content/plugins/ folder on your server.
WordPress: Navigate to Plugins -> Add New-> Upload page. Select the lemur-slider.zip file. Press the Install Now button to upload and install the plugin.
After uploading the plugin, you have to activate it. Navigate to Plugins -> Installed Plugins, look for the Lemur Slider plugin and click Activate.
